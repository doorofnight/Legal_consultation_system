# 试用期违法解除劳动合同案例分析

## 案例基本信息

- **案号**：(2023)京0105民初12345号
- **案件类型**：劳动争议
- **审理法院**：北京市朝阳区人民法院
- **判决时间**：2023年6月15日

## 案情简介

原告张某于2022年12月1日入职被告某科技公司，担任前端开发工程师，劳动合同期限为3年，约定试用期为6个月，试用期工资为8000元，转正后工资为10000元。

2023年3月15日，公司在试用期内以"不符合录用条件"为由解除与张某的劳动合同。张某认为公司解除行为违法，向劳动仲裁委员会申请仲裁，要求支付违法解除劳动合同赔偿金。

## 争议焦点

1. 公司解除劳动合同的理由是否成立？
2. 公司是否履行了证明劳动者不符合录用条件的举证责任？
3. 解除程序是否合法？

## 法院查明事实

1. **录用条件**：公司提供的录用条件仅为"具备前端开发能力"，未明确具体标准。
2. **工作表现**：张某在职期间完成了公司分配的工作任务，无重大工作失误。
3. **解除程序**：公司未提供证据证明张某不符合录用条件，也未在解除前向张某说明理由。
4. **解除通知**：公司通过微信单方面通知解除，未出具书面解除证明。

## 法律适用

### 《中华人民共和国劳动合同法》相关规定：

1. **第二十一条**：在试用期中，除劳动者有本法第三十九条和第四十条第一项、第二项规定的情形外，用人单位不得解除劳动合同。用人单位在试用期解除劳动合同的，应当向劳动者说明理由。

2. **第三十九条**：劳动者在试用期间被证明不符合录用条件的，用人单位可以解除劳动合同。

3. **第四十八条**：用人单位违反本法规定解除或者终止劳动合同，劳动者要求继续履行劳动合同的，用人单位应当继续履行；劳动者不要求继续履行劳动合同或者劳动合同已经不能继续履行的，用人单位应当依照本法第八十七条规定支付赔偿金。

## 法院判决

### 判决结果：

1. 确认被告公司解除劳动合同行为违法。
2. 被告公司向原告张某支付违法解除劳动合同赔偿金20000元（计算方式：经济补偿标准的二倍）。
3. 被告公司为原告张某出具解除劳动合同证明。

### 判决理由：

1. **举证责任**：用人单位以"不符合录用条件"解除试用期员工，负有举证责任。被告公司未能提供证据证明原告不符合录用条件。
2. **录用条件明确性**：被告公司的录用条件过于模糊，不具备可操作性。
3. **解除程序**：被告公司未履行说明理由的法定义务，解除程序违法。
4. **赔偿计算**：原告工作3.5个月，月工资10000元，经济补偿为5000元（10000元×0.5个月），赔偿金为10000元（5000元×2倍）。

## 案例启示

### 对用人单位的启示：

1. **明确录用条件**：录用条件应当具体、明确、可衡量。
2. **完善考核制度**：建立试用期考核制度，定期评估员工表现。
3. **规范解除程序**：解除前应当告知理由，给予申辩机会。
4. **保留证据**：保存员工不符合录用条件的相关证据。

### 对劳动者的启示：

1. **了解权利**：试用期员工享有与正式员工同等的法律保护。
2. **保留证据**：保存工作记录、考核结果等证据。
3. **依法维权**：遇到违法解除，及时通过法律途径维权。

## 风险防范建议

1. **企业合规建议**：
   - 制定明确的岗位职责和录用标准
   - 建立规范的试用期考核流程
   - 完善解除劳动合同的内部审批程序

2. **员工权益保护**：
   - 仔细阅读劳动合同和录用条件
   - 定期与上级沟通工作表现
   - 遇到问题及时咨询法律专业人士
