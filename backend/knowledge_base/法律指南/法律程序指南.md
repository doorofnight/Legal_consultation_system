# 法律程序指南

## 第一章 劳动仲裁程序

### 1.1 申请条件
1. 存在劳动争议；
2. 属于劳动仲裁受理范围；
3. 在仲裁时效内（自知道或者应当知道权利被侵害之日起一年内）。

### 1.2 申请材料
1. 劳动仲裁申请书（一式三份）；
2. 申请人身份证明；
3. 与被申请人存在劳动关系的证明材料；
4. 其他与案件有关的证据材料。

### 1.3 仲裁流程
1. 提交申请：向有管辖权的劳动人事争议仲裁委员会提交仲裁申请；
2. 受理审查：仲裁委员会在收到申请后5日内决定是否受理；
3. 组成仲裁庭：仲裁委员会指定仲裁员组成仲裁庭；
4. 开庭审理：仲裁庭在开庭5日前将开庭日期、地点书面通知双方当事人；
5. 调解：仲裁庭在作出裁决前应当先行调解；
6. 裁决：调解不成的，仲裁庭应当及时作出裁决。

### 1.4 仲裁裁决
1. 仲裁裁决书应当载明以下内容：
   - 申请人和被申请人的基本情况；
   - 仲裁请求和争议事实；
   - 裁决理由和裁决结果；
   - 裁决日期。
2. 仲裁裁决为终局裁决，自作出之日起发生法律效力。

## 第二章 民事诉讼程序

### 2.1 起诉条件
1. 原告是与本案有直接利害关系的公民、法人和其他组织；
2. 有明确的被告；
3. 有具体的诉讼请求和事实、理由；
4. 属于人民法院受理民事诉讼的范围和受诉人民法院管辖。

### 2.2 起诉材料
1. 起诉状（一式三份）；
2. 原告身份证明；
3. 与案件有关的证据材料。

### 2.3 诉讼流程
1. 立案：人民法院在收到起诉材料后7日内决定是否立案；
2. 审理：人民法院组成合议庭或由审判员独任审理；
3. 判决：人民法院在法定期限内作出判决。
