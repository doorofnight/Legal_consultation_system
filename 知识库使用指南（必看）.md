## 📁 知识库使用指南

所有ai功能都有知识库的参与，想要使用知识库必须对知识库向量化

### 知识库向量化
#### 方式1：通过批处理脚本一键启动(Windows，推荐)
```bash
cd backend
init_vector_store.bat
```

#### 方式2：通过API接口
```bash
# 扫描文档
curl -X POST http://localhost:8000/api/v1/knowledge/scan

# 向量化文档（使用Ollama）
curl -X POST "http://localhost:8000/api/v1/knowledge/vectorize?model_provider=ollama"

# 重建向量存储
curl -X POST "http://localhost:8000/api/v1/knowledge/vectorize?rebuild=true&model_provider=ollama"
```

###  知识库添加新文档

### 方式1：前端添加新文档
1. 访问知识库管理页面 (http://localhost:5173/knowledge)
2. 点击"上传文档"按钮
3. 选择文件上传
4. 系统会自动处理文档并添加到知识库

### 方式2：后端直接添加
1. 将文档文件放入 `Legal_consultation_system/backend/knowledge_base` 文件夹中
2. 支持的文件格式：`.md`, `.txt`, `.pdf`, `.docx`
3. 运行向量化脚本或调用API重新构建向量存储
